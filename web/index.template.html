🚌 퇴근 버스 도착정보

상록마을에서 미금역을 거쳐 집으로 가는 버스만 표시합니다.

인증키는 기본 입력되어 있으며 화면에서 마스킹됩니다. 자동 갱신은 페이지가 화면에 보일 때만 작동합니다.
API 및 인증키 안내

이 페이지는 공공데이터포털에서 제공하는 경기도 버스 Open API를 사용합니다.

사용 API

  • 경기도 버스도착정보 조회 — 정류소별 실시간 도착 예정 시간
  • 경기도 버스노선 조회 — 노선 ID를 버스 번호로 변환
  • 경기도 정류소 조회 — 표기 정류소 번호를 내부 ID로 변환

인증키 발급 방법

  1. 공공데이터포털에 로그인합니다.
  2. 위 API 세 개를 각각 열고 활용신청을 누릅니다.
  3. 마이페이지의 Open API 개발계정에서 승인 상태를 확인합니다.
  4. 발급된 일반 인증키 중 Decoding 키를 인증키 입력란에 사용합니다.

개발계정의 기본 신청 가능 트래픽은 API별 하루 1,000회입니다. 이 페이지는 기본 수동 조회이며 자동 갱신은 10분으로 제한합니다.
